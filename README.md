# 🌿 Agro Forte

Uma landing page moderna sobre sustentabilidade e inovação agrícola desenvolvida com HTML e CSS.

---

## 📌 Descrição

O projeto **Agro Forte** foi criado com o objetivo de apresentar uma interface simples, elegante e responsiva voltada ao tema de agricultura sustentável.

A aplicação contém:

* Página institucional
* Banner principal
* Sessões organizadas
* Cards informativos
* Layout responsivo
* Design moderno em tons verdes

---

# 🛠️ Tecnologias Utilizadas

| Tecnologia    | Descrição                        |
| ------------- | -------------------------------- |
| HTML5         | Estrutura da página              |
| CSS3          | Estilização e responsividade     |
| Flexbox       | Organização dos elementos        |
| Media Queries | Ajustes para dispositivos móveis |
